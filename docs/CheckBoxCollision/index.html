CheckBoxCollision

CheckBoxCollision is a function that returns a boolean if 2 given rectangles are inside of each other.


Imagine a variable which value ranges from 0 to 100, and you would like to represent that as a color. In order to show the full range of the value, the color that you will be showing goes from 0 to 255.

The remap function calculates this for you!

The function has 5 parameters; value, from1, to1, from2, to2, in that order. From1 and to1 are the minimum and maximum values the value can be, same for from2 and to2, but this time for the output value.

For the example mentioned above, it would be:
let outputValue = Remap(value, 0, 100, 0 255);
If the input value is 50, the output would be 127,5.
